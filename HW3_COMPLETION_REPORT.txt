### HW3b: Wave Animation with Multiple Surface Geometries

**Requirements (From Assignment):**
- Display modes: Wireframe, Textured, Textured+Wireframe ✓
- Ten different surface geometries ✓
- Grid size slider (2-64) ✓
- Animation start/stop button ✓
- Fill skeletal code marked with "PUT YOUR CODE HERE" ✓

**Implementation Details:**

1. **resizeGL()** - Perspective projection setup
   - 45-degree field of view
   - Dynamic aspect ratio calculation
   - Proper near and far clipping planes

2. **resetMesh() - Geometry Implementations:**
   - FLAT: Flat surface
   - SPIKE: Single central spike
   - HOLE: Circular depression in center
   - DIAGONALWALL: Diagonal wall structure
   - SIDEWALL: Vertical wall through center
   - DIAGONALBLOCK: Diagonal striped pattern
   - MIDDLEBLOCK: Raised central block
   - CORNERBLOCK: Four corner blocks
   - HILL: Smooth gaussian hill
   - HILLFOUR: Four gaussian hills

3. **paintGL() - Display Mode Rendering:**
   - TEXTURED_WIREFRAME: Renders both texture and wireframe
   - TEXTURED: Texture only
   - WIREFRAME: Wireframe only
   - FLAT_COLOR: Flat shading with colors
   - SMOOTH_COLOR: Smooth shading with colors
   - SMOOTH_TEXTURE: Smooth shading with texture

**Features:**
- 10 different geometric surfaces
- 6 display modes
- Grid resolution from 2x2 to 64x64
- Wave animation with adjustable speed
- Interactive camera and light control
- Proper normal calculations for shading
